# 🏹 ArrowFlow (Neon Puzzle Escape)

ArrowFlow is a sleek, modern, cyberpunk-inspired 2D puzzle game built using clean HTML, CSS, and JavaScript. Players navigate glowing neon arrows through challenging grids to find the ultimate escape path. 

With an immersive dark aesthetic and thousands of engaging levels, it tests your logic, strategy, and spatial awareness in a minimalist digital environment.

---

## 🚀 Features

* **5,000 Unique Levels:** From beginner-friendly grids to mind-bending expert challenges.
* **Cyberpunk Aesthetic:** High-contrast neon glows, beautiful dark mode interface, and clean animations.
* **Smooth Controls:** Intuitive "Previous", "Reset", and "Next" navigation panel for seamless gameplay.
* **Lightweight & Fast:** Pure vanilla web technologies ensure instant loading and zero lag on mobile and web devices.

---

## 🛠️ Tech Stack

* **HTML5:** Game structure and UI layout.
* **CSS3:** Neon glow effects, grid styling, and responsive design.
* **JavaScript (ES6):** Game logic, level generation, and control systems.

---

## 🎮 How to Play

1. Open the game in any mobile or desktop browser.
2. Analyze the grid layout and the direction of the neon arrows.
3. Use the navigation controls to solve the puzzle and escape the current level.
4. Progress through the levels and challenge your high score!
